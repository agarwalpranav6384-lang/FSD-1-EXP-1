Student Registration Form
This project is a simple Student Registration Form developed using HTML, CSS, and JavaScript. It provides a clean and user-friendly interface for users to enter their details such as full name, email, and age, along with proper input validation. The form ensures that the name contains only letters and is between 9–50 characters, the email follows a valid format, and the age is restricted between 16 and 30. If any input is incorrect, error messages are displayed with highlighted fields, and a success message appears when all validations pass. The project consists of three files: index.html for structure, style.css for styling, and script.js for validation logic. It can be easily run by opening the index.html file in any web browser and serves as a basic example of front-end form validation.

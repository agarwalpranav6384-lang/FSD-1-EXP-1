This project is a beginner-friendly dashboard developed using HTML and CSS to help students understand the basics of web design and layout creation. It features a clean and simple user interface that includes a header, a sidebar navigation menu, and a main content area displaying basic information cards. The project focuses on core frontend concepts such as page structure, styling, alignment, spacing, and responsive layout using CSS Flexbox. No JavaScript is used, making the code easy to understand, modify, and practice. This dashboard is ideal for beginners, college students, and anyone starting their journey in web development.
